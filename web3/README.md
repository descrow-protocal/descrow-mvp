# Descrow Smart Contracts

## Setup

```bash
npm install
cp .env.example .env
# Edit .env with your PRIVATE_KEY
```

## Compile

```bash
npm run compile
```

## Deploy to Moonbeam Alpha

```bash
npm run deploy
```

## Network Details

- **Chain ID**: 1287
